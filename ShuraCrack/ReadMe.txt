Цель задания - взломать программу CrackMe.com. 

Реализовано 2 вида взлома: 

1) Через ввод слишком большого пароля, дабы переполнить буфер и записать нужное нам за его пределами

	Для реализации этого вида взлома нужно запустить CrackMe.com с вводом из Hack.txt
	
	CrackMe.com < Hack.txt

2) Через создание кряк-версии CrackMe.com с помощью программы Crack.exe

	Для этого нужно запустить Crack.exe и подать ему на ввод CrackMe.com. После работы программы создастся Cracked.com, которая не будет требовать пароля, а сразу выведет "Fine, you entered successfully."
